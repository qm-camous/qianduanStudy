jQuery - 删除元素

一、简介

删除元素/内容： 如需删除元素和内容，一般可使用以下两个 jQuery 方法：
remove() - 删除被选元素（及其子元素） empty() - 从被选元素中删除子元素

二、jQuery remove() 方法

jQuery remove() 方法删除被选元素及其子元素。

三、jQuery empty() 方法

jQuery empty() 方法删除被选元素的子元素。

此水几时休，此恨何时已。
