栅格系统简介：

栅格系统必须要通过container承载，container容器包括class="container"和class="container-fluid"两个类，后者占用整个视口的整个面积，实现最大化，通常使用class="container"即可。

栅格系统最多将系统分为12列，通过行和列布局

参数

都是12列，槽宽30px。可嵌套可偏移可排序。

1.分比率小于768px（手机）。在手机上水平排列，会自动堆放。宽度自适应。前缀".col-xs-"

2.分比率大于768px（主要是PAD）。宽度最大750px。前缀".col-sm-"

3.分比率大992px（主要是小屏幕的多媒体显示器）。宽度最大970px,前缀".col-md-"

4.分比率大1200px（主要是大屏幕的多媒体显示器）。宽度最大1170px,前缀".col-lg-"
